1. **授权阶段**：用户通过`union approve`合约授权数字资产
2. **聚合交互**：调用`Aggregator`合约接口执行交易
3. **市场对接**：通过`Adapter`适配器合约连接各市场

```markdown
[用户] → [Aggregator合约] → [Adapter合约] → [目标市场]
```
